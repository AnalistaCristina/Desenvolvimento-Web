# Lumière Beauté — Clínica de Estética

Sistema web completo para clínica de estética com design feminino, moderno e elegante.

## Estrutura de Arquivos

```
lumiere-beaute/
├── index.html        ← Arquivo principal (abrir no navegador)
├── css/
│   └── style.css     ← Estilos e tema visual
├── js/
│   └── app.js        ← Navegação e interatividade
└── README.md         ← Este arquivo
```

## Como usar

1. Extraia o arquivo `.zip`
2. Abra o arquivo `index.html` em qualquer navegador moderno
3. Não é necessário servidor — funciona localmente

## Páginas disponíveis

| Página       | Descrição                                              |
|--------------|--------------------------------------------------------|
| Início       | Hero de boas-vindas, diferenciais e estatísticas       |
| Quem Somos   | História da clínica e apresentação da equipe           |
| Serviços     | 4 tratamentos com descrição, duração e preço           |
| Contato      | Informações de contato, horários e formulário          |

## Personalização

- **Nome / logotipo**: edite `index.html` — busque por "Lumière Beauté"
- **Cores**: edite as variáveis CSS no início de `css/style.css` (seção `:root`)
- **Serviços**: edite os cards em `#page-servicos` dentro de `index.html`
- **Informações de contato**: edite a seção `#page-contato` em `index.html`
- **Fontes**: substituídas via Google Fonts no `<head>` de `index.html`

## Tecnologias

- HTML5 semântico
- CSS3 com variáveis e animações
- JavaScript vanilla (sem frameworks)
- Google Fonts: Cormorant Garamond + DM Sans
